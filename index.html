Introduction to HTML

HTML stands for HyperText Markup Language and is used to create the structure and content of a webpage.
Most HTML elements contain opening and closing tags with raw text or other HTML tags between them.
HTML elements can be nested inside other elements. The enclosed element is the child of the enclosing parent element.

HTML Document Standards

Jump to Top

While some tags have a very specific purpose, such as image and video tags, most tags are used to describe the content that they surround, which helps us modify and style our content later. There are seemingly infinite numbers of tags to use. Knowing when to use each one is based on how you want to describe the content of your HTML. Descripte, well-chosen tags are one key to high-quality web development. A full list of available HTML tags can be found in Mozilla documentation.

Whitespace between HTML elements helps make code easier to read while not changing how elements appear in the browers. Indentation also helps make code easier to read, it makes parent-child relationships visible.

HTML Document Standards
Tag Name Description
<!DOCTYPE html> Documentation Declaration for HTML This declaration should always be the first line of code in your HTML files. This lets the browser know what version of HTML to expect.
<html> HTML Element This element contains all of your HTML code
<head> Head Element Information about the web page, like the title, belongs within the head of the page.
<title> Title Element You can add a title to your web page by using the title element, inside of the head.
A webpage's title appears in a browser's tab.
<!-- comment --> Comment Syntax Using this syntax, comments can be written in HTML.
